## 💡 ASTUCES MALIN

### Comment TROUVER un prix à modifier:
1. Ouvrir index.html
2. Appuyer Ctrl+F
3. Taper: "2500"
4. Voir tous les prix qui s'affichent
5. Double-cliquer pour modifier

### Comment TESTER rapidement:
1. Faire un changement dans index.html
2. Sauvegarder (Ctrl+S)
3. Aller au navigateur
4. Appuyer F5
5. Voir le changement ✓

### Comment VOIR LES ERREURS:
1. Appuyer F12
2. Aller à "Console"
3. Les erreurs sont en rouge
4. Les logs sont en bleu

---

## 🛑 SI QUELQUE CHOSE NE FONCTIONNE PAS

### Le site ne s'ouvre pas:
→ Vérifiez que index.html existe
→ Essayez avec un autre navigateur
→ Utilisez Chrome si possible

### Les images ne s'affichent pas:
→ Vérifiez les fichiers sont dans images/
→ Vérifiez les noms sont corrects
→ Appuyez F5 pour rafraîchir

### Le panier ne fonctionne pas:
→ Ouvrir F12 → Console
→ Voir si message d'erreur rouge
→ Vérifier que JavaScript est activé

### Le formulaire ne valide pas:
→ Tous les champs * doivent être remplis
→ Le téléphone doit avoir 8 chiffres minimum
→ L'email doit avoir format valide

### Le site s'affiche mal sur mobile:
→ C'est normal (images placeholder)
→ Ça sera mieux avec vos vraies images
→ Tester quand images ajoutées
