# Mettre OKOKU en ligne

## Option rapide avec Netlify

1. Va sur https://app.netlify.com/drop
2. Glisse le dossier `JS` complet dans la page.
3. Netlify publie le site et donne une adresse publique.

## Option avec Vercel

1. Va sur https://vercel.com/new
2. Importe ce dossier comme projet statique.
3. Laisse les reglages par defaut :
   - Framework Preset: Other
   - Build Command: vide
   - Output Directory: `.`

## Fichiers importants

- `index.html`
- `style.css`
- `script.js`
- `image.png`
- `images/`

Il faut envoyer tout le dossier pour que les images s'affichent correctement.
