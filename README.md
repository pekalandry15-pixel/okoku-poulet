## ❓ QUESTIONS FRÉQUENTES

**Q: Les images ne s'affichent pas?**
A: Vérifiez que les fichiers sont dans le dossier `images/` avec les bons noms

**Q: Comment changer le prix?**
A: Ouvrir `index.html` → Chercher le prix → Remplacer

**Q: Le panier ne fonctionne pas?**
A: Ouvrir F12 (Console) → Vérifier les erreurs

**Q: Comment tester sur mobile?**
A: Appuyez F12 → Icône mobile → Choisir votre téléphone

**Q: Comment ajouter un produit?**
A: Copier un bloc `<div class="product-card">` → Modifier → Sauvegarder
