```bash

```

### Option 3: Serveur Local Node.js
```bash
# Installer http-server:
npm install -g http-server

# Lancer le serveur:
http-server

# Ouvrir: http://localhost:8080
```

## 📸 AJOUTER VOTRE MARQUE

### Pour remplacer les images:

1. **Logo OKOKU**
   - Dimension: 200x200 pixels minimum
   - Format: PNG (transparence) ou JPG
   - Fichier: `images/logo-okoku.png`

2. **Photo Restaurant**
   - Dimension: 800x600 pixels
   - Format: JPG ou PNG
   - Fichier: `images/okoku-restaurant.jpg`

3. **Photos Plats**
   - Dimension: 600x400 pixels
   - Format: JPG
   - Fichier: `images/poulet-plats.jpg`

### Étapes:
1. Prenez vos photos (vous en avez déjà 2!)
2. Redimensionnez-les si besoin
3. Renommez-les exactement (voir ci-dessus)
4. Collez-les dans le dossier `images/`
5. Rafraîchissez la page (F5)

## ⚙️ PERSONNALISATION RAPIDE

### Changer le prix d'un produit:
Ouvrir `index.html` avec Notepad++/VSCode
Chercher: `addToCart('Poulet Grillé', 2500)`
Changer `2500` par le nouveau prix

### Ajouter votre numéro de téléphone:
Chercher: `📞 +237 6XX XXX XXX`
Remplacer par votre vrai numéro

### Changer les horaires:
Chercher: `⏰ <strong>Lundi - Dimanche: 10h - 23h</strong>`
Remplacer par vos horaires réels

## 🧪 TESTER LE SITE

### Test du panier:
1. Cliquez plusieurs fois "Ajouter" sur des produits
2. Allez à l'onglet "Panier"
3. Vérifiez que le total est correct
4. Retirez un produit
5. Le total se recalcule ✓

### Test du formulaire:
1. Allez à "Télécommander"
2. Remplissez les champs (Nom, Téléphone, Adresse)
3. Cliquez "Valider la Commande"
4. Vous voyez la confirmation ✓
5. Le formulaire se réinitialise ✓

### Test sur mobile:
1. Appuyez sur F12 (Console développeur)
2. Cliquez l'icône "Mobile" (ou Ctrl+Shift+M)
3. Testez sur iPhone, Pixel, Tablet
4. Vérifiez que tout s'affiche bien ✓
